# Oasis Infobyte Data Analytics Internship

This repository contains all the projects completed during the Oasis Infobyte Data Analytics Internship using Python and Data Analytics libraries.

---

## Projects Included

### 1. EDA on McDonald’s Nutrition Dataset
Performed Exploratory Data Analysis (EDA) on McDonald’s nutrition dataset to analyze calories, fat, sugar, protein, and nutritional values using visualizations and statistical analysis.

### 2. Customer Segmentation Analysis
Analyzed customer purchasing behavior and customer-related data using Python and visualization techniques to generate meaningful business insights.

### 3. NYC Airbnb Data Cleaning
Performed data cleaning and preprocessing on NYC Airbnb Open Dataset by handling missing values, duplicates, outliers, and datatype inconsistencies.

### 4. Twitter Sentiment Analysis
Analyzed Twitter sentiment data to study positive, negative, and neutral sentiments using data cleaning, visualization, and exploratory analysis techniques.

### 5. Google Play Store Data Analysis
Analyzed Google Play Store app data to explore app categories, ratings, installs, and pricing trends using visualizations and statistical analysis.
